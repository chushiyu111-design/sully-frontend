# 粉蓝云边纸条聊天主题

一套给 `气泡工坊 -> CSS 增强模式` 使用的低饱和贴纸纸条气泡主题。

这版重点不是复刻某张背景图，而是做现在更常见的“年轻感气泡”：白色纸条底、浅紫浅蓝描边、柔光边缘、手写感文字和云朵小角标。整体保持粉蓝低饱和，长期聊天不会刺眼。

## 文件

- `pastel-cloud-note.css`：复制到气泡工坊的完整 CSS。
- `preview.html`：本地预览页，方便快速检查文字、语音、卡片、图片消息和底栏效果。
- `README.md`：使用说明、推荐设置和发布文案。

## 使用步骤

1. 进入应用里的 `气泡工坊`。
2. 新建主题，切到 `CSS` 标签。
3. 整段复制 `pastel-cloud-note.css`，粘贴到 `CSS 增强模式`。
4. 保存主题，在 Message 聊天页切换到这个 DIY 主题。

本地开发时可以打开：

```text
http://localhost:5173/community-bubble-themes/pastel-cloud-note/preview.html
```

## 可替换变量

CSS 顶部只有一个可选背景变量：

```css
--pcn-chat-bg: none;
```

如果你想加自己的浅色背景图，可以改成：

```css
--pcn-chat-bg: url("你的图床直链");
```

不想加背景就保持 `none`。气泡本身不依赖图片。

## 气泡工坊推荐基础设置

CSS 会尽量覆盖视觉，但为了真实聊天页和工坊预览都稳定，建议先这样调：

- 主题名称：`粉蓝云边纸条`
- 用户气泡背景色：`#fff7fd`
- 角色气泡背景色：`#f4fbff`
- 用户/角色文字颜色：`#6d6f80`
- 圆角：`12` 到 `16`
- 透明度：`1`
- 气泡紧凑度：`适中`
- 不额外上传气泡底纹，纸条和云角标由 CSS 生成

## 覆盖范围

这份 CSS 会覆盖：

- 聊天背景轻底色
- 顶部栏
- 底部输入栏
- 文字气泡
- 语音气泡
- 打字中气泡
- 系统提示 pill
- 互动 pill
- 图片消息
- 转账卡片
- 卡片类消息
- 常见按钮和翻译按钮

## 常见问题

**为什么看起来没有截图那么花？**
这版只参考“纸条气泡”的趋势，不复刻截图里的背景和具体角色图案，避免变成只能配一张背景的皮肤。

**为什么字体不像截图手写体？**
CSS 使用系统字体兜底，比如 `KaiTi`、`STKaiti`、`Segoe Print`。不同设备会略有差异；如果设备没有手写字体，会自动回退到普通字体。

**会不会遮住文字？**
云朵角标是 CSS 生成的，并放在气泡角落外侧，文字层级更高。窄屏下角标会自动缩小。

**能不能换成自己的背景？**
可以，把 `--pcn-chat-bg` 改成图床直链即可。不换也能用。

**会不会影响其他页面？**
CSS 是粘贴在某个 DIY 主题里的，只有该主题被聊天页启用时才会注入。选择器主要限制在 `sully-chat-*` 和 `sully-*` 聊天组件类名上。

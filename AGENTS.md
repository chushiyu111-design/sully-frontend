# Repo Agent Rules

本仓库是前端仓库，负责页面、组件、样式、浏览器侧逻辑与前端代理调用。

## 进入本仓库前先遵守

- 优先遵守上级工作区的编码规则；如果你是单独打开本仓库，也必须执行下面的字符编码规则
- 所有新增或修改的文本文件默认使用 UTF-8
- `*.ps1`、`*.psm1`、`*.psd1` 如果包含非 ASCII 字符，使用 UTF-8 with BOM
- 文本读写显式指定 UTF-8，不依赖 Windows 默认编码
- 修改中文内容后，重新打开文件确认没有乱码
- 如果只是在终端里看到乱码，先判断终端编码，不要立刻覆盖文件

## 仓库范围

- React、Vite、Tailwind、页面结构、前端配置、前端调用链路，都在本仓库内处理
